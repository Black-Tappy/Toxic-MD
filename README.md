## ‚ö° Quick Start

> **Note:** This script is not deployable on VPS/panel or Termux. Only Render, Heroku, and Replit are supported.

1. **Get Session ID:**
    ```bash
    Visit: https://toxic-pair-code.onrender.com
    Scan QR via WhatsApp > Linked Devices
    ```
